SignalSmith AI – Adaptive Modulation Scheme Predictor

SignalSmith AI is a machine learning-powered modulation scheme prediction system that intelligently selects the optimal digital modulation technique based on wireless channel conditions. Using an XGBoost classifier trained on parameters such as Signal-to-Noise Ratio (SNR), channel type, and Doppler frequency, the system predicts the most suitable modulation scheme (BPSK, QPSK, or QAM) to maximize communication reliability and efficiency.

The project includes a complete ML pipeline for data preprocessing, model training, evaluation, and deployment. The trained model is served through a Flask REST API, enabling real-time predictions and seamless integration with wireless communication simulations, research projects, and networking applications.

Key Features

Adaptive modulation prediction using XGBoost
Classification of BPSK, QPSK, and QAM schemes
End-to-end machine learning pipeline with Scikit-learn
Real-time prediction through Flask REST API
Modular and deployment-ready architecture
Built with Python, Pandas, NumPy, Joblib, and Flask

Tech Stack: Python, XGBoost, Scikit-learn, Pandas, NumPy, Flask, Joblib, REST API, Machine Learning.
